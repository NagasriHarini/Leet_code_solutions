# Sort Colors (LeetCode 75)

## Problem Statement

Given an array `nums` containing only `0`, `1`, and `2`, sort the array in-place so that objects of the same color are adjacent, with the colors in the order:

```
0s → 1s → 2s
```

You must solve the problem without using the library sort function.

### Example

**Input**

```python
nums = [2,0,2,1,1,0]
```

**Output**

```python
[0,0,1,1,2,2]
```
