# Pascal's Triangle (LeetCode 118)

## Problem Statement

Given an integer `numRows`, return the first `numRows` rows of Pascal's Triangle.

In Pascal's Triangle, each number is the sum of the two numbers directly above it.

---

## Example 1

### Input

```text
numRows = 5
```

### Output

```text
[[1],
 [1,1],
 [1,2,1],
 [1,3,3,1],
 [1,4,6,4,1]]
```

---

## Example 2

### Input

```text
numRows = 1
```

### Output

```text
[[1]]
```

---

## Constraints

```text
1 <= numRows <= 30
```
