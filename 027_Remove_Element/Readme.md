# Remove Element (LeetCode 27)

## Problem Statement

Given an integer array `nums` and an integer `val`, remove all occurrences of `val` in-place.

The order of the remaining elements may be changed.

Return the number of elements remaining after removal (`k`).

The first `k` elements of `nums` should contain the elements that are not equal to `val`.

You must perform the operation using **O(1)** extra space.

---

## Example 1

**Input**

```text
nums = [3,2,2,3]
val = 3
```

**Output**

```text
2
```

**Modified Array**

```text
[2,2,_,_]
```

---

## Example 2

**Input**

```text
nums = [0,1,2,2,3,0,4,2]
val = 2
```

**Output**

```text
5
```

**Modified Array**

```text
[0,1,3,0,4,_,_,_]
```
